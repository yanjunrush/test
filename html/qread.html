快速读入




在c++中 cin 的读入效率是很低很低的，一般我们可以用scanf， 但是在读取较多较大的数字时，这两个都会花掉我们很多时间，所以我们可以用快速读入(貌似只能读数字TAT) 输入字符转换成数字，效率高，在输入数据量特别大的时候采用快速读入可以避免超时 当然正负可以保证。如图左







也可以这样写：

                                                            inline int read()
                                                            {
                                                                int k=0;
                                                                char f=1;
                                                                char c=getchar();
                                                                for(;!isdigit(c);c=getchar() )
                                                                    if(c=='-')
                                                                        f=-1;
                                                                for(;isdigit(c);c=getchar() )
                                                                    k=k*10+c-'0';
                                                                return k*f;
                                                            }
